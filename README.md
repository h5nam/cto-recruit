# Mindful Labs — CTO Recruitment Terminal

상담사는 오직 상담에만. 심리 상담 산업의 OS를 함께 만들 CTO를 찾습니다.

## Overview

마인드풀랩스 CTO 채용을 위한 인터랙티브 터미널 스타일 웹페이지입니다. 방문자가 터미널 명령어를 입력하거나 Enter 키를 눌러 채용 정보를 순차적으로 탐색할 수 있습니다.

## Live

GitHub Pages 또는 정적 호스팅으로 `index.html`을 배포하면 바로 사용할 수 있습니다.

## Sections

| # | Command | Section |
|---|---------|---------|
| 0 | `cat the_irony.md` | 시대의 아이러니, 그리고 우리의 반격 |
| 1 | `cat product.md` | 마음토스: 라이브 프로덕트 |
| 2 | `cat the_masterplan.md` | 비전과 마스터플랜 |
| 3 | `cat challenges.md` | 주요 업무와 기술적 도전 |
| 4 | `cat requirements.md` | 자격 요건 및 보상 |
| 5 | `./join_mindfullabs.sh` | 합류하기 |

## Tech Stack

- HTML5 single-file application
- Tailwind CSS (CDN)
- Vanilla JavaScript (ES6+)
- JetBrains Mono + Pretendard Variable fonts
